# Financial-Fraud-Detection-
A Java-based fraud detection system that uses rule-based heuristics and multithreading to analyze transactions in real time, with a GUI based on Swing.
The system analyzes transaction attributes such as amount, frequency, location, and time to classify transactions as Normal, Suspicious, or Fraud.
It features a desktop-based graphical user interface built using Java Swing, allowing users to visualize transaction data and fraud alerts in real time. The application simulates live transaction processing using multithreading with ExecutorService, ensuring efficient and responsive performance.
The system is structured using a layered architecture (Model, DAO, Service, UI), promoting modularity, scalability, and maintainability. Data is managed through CSV files, enabling lightweight storage and easy integration without requiring a database.
